# Xyvex

AI that learns the operator.

## Vision
Xyvex is a personal intelligence system designed to:
- track habits and behavior
- analyze productivity patterns
- optimize goals and workflows
- provide personalized recommendations
- evolve with long term memory

Unlike other AI powered assistants, Xyvex improves understanding of you over time.

## Features
- AI powered reflections
- Goal intelligence
- Behavioral analytics
- Long term memory
- Trend analysis
- Personalized optimization
- Adaptive recommendations

## Tech Stack
- Next.js
- FastAPI
- PostgreSQL
- ChromaDB
- OpenAI API
